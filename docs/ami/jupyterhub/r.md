# R

The JupyterHub AMI comes with multiple R versions plus some default
data science packages installed out of the box.

## R versions

| Version   | Location       |
| --------- | -------------- |
| `R 4.1.2` | `/opt/R/4.1.2` |
| `R 4.0.3` | `/opt/R/4.0.3` |
| `R 3.6.3` | `/opt/R/3.6.3` |

!!! info "Owner"

    The owner of this directories is the `ubuntu` user (the same one used to SSH).
